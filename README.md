This portfolio serves as an interactive experience, featuring:

* **Design Philosophy:** Cyberpunk-inspired "Lab Terminal" interface with glassmorphism and real-time telemetry effects.

- **Data Analytics:** Pandas, NumPy, Matplotlib, Power BI
- **Deployment:** GitHub Pages, Vercel
